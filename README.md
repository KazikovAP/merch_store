## Запуск проекта

**1. Клонировать репозиторий:**
```
git clone https://github.com/KazikovAP/merch_store.git
```

**2. Сборка и запуск проекта:**
```
docker-compose up --build
```

**3. Остановка и удаление контейнеров:**
```
docker-compose down
```
